bcc INSTALL
===========

Installation instructions.

Requirements
------------

- C99, a C99 compiler (GCC, Clang),
- Standard make,
- Some POSIX functionalities (getopt).

Basic installation
------------------

Quick install.

	$ bsdtar -xvf bcc-x.y.z.tar.xz
	$ cd bcc-x.y.z
	$ make
	# sudo make install
