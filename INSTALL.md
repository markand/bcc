bcc INSTALL
===========

Installation instructions.

Requirements
------------

- C11, a C11 compiler (GCC, Clang),
- POSIX make, a POSIX make implementation,
- POSIX getopt, the utility make use of `getopt(3)`.

Basic installation
------------------

Quick install.

	$ tar xvzf bcc-x.y.z-tar.xz
	$ cd bcc-x.y.z
	$ make
	# sudo make install
